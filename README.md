# railway-reservation-system

## Project Description
This is a simple Railway Reservation System developed using Python.  
It allows users to book, view, and cancel tickets.

## Features
- Check seat availability
- Book tickets
- View ticket details using Booking ID
- Cancel tickets
- Simple menu-driven interface

## Technologies Used
- Python

## How to Run
1. Download or clone the repository
2. Open terminal / command prompt
3. Run:
   python main.py
